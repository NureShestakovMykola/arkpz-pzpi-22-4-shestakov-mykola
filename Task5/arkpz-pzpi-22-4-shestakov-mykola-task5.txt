1 МЕТА

	Метою роботи є розгорнути програмну систему та продемонструвати її роботу.

2 ХІД РОБОТИ

	Виконання лабораторної роботи включало:
    1. Розгортання програмної системи для автоматичного догляду за рослинами GrowMate за допомогою засобів Visual Studio на локальному сервері;
    2. Додавання функціональності створення пустої бази даних із використанням засобів Entity Framework до бізнес-логіки серверної частини системи;
    3. Створення додаткового bat файлу для запуску програмної системи на необхідному порту, який до цього було відкрито для доступу з мережі за допомогою dev tunnel.
Загальний опис фінальної програмної системи:
    • Мови програмування: C# (сервер) та C++ (IoT девайс);
    • Сервер: Asp.Net Core;
    • База даних: локальна, SQL Server;
    • IoT пристрій: включає сенсор вологості та температури, кнопку для увімкнення та вимикання, вбудований в контролер Wi-Fi модуль та світлодіод для імітації роботи поливного механізму;
    • Середи розробки: Visual Studio (сервер), https://wokwi.com (IoT девайс);
    • Метод розгортання: публікація готового архіву з файлом засобами Visual Studio, розгортання на локальному хості.

ВИСНОВКИ

Протягом лабораторної системи було розгорнуто програмну систему для автоматичного догляду за рослинами GrowMat а також пробемонстровано її роботу.
